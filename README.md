# smishing 

강의를 위한 스미싱을 차단하기 위해한 서버 및 안드로이드 앱 개발.

#### To build
```
docker build -t smishing .
```


#### To run
```
docker run -d --name mycontainer -p 80:8080 smishing
```


### Build & run docker-compose
```
docker-compose up -d 
or
docker-compose up -d --build   # for rebuild and running.
```
